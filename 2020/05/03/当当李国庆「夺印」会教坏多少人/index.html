当当李国庆「夺印」会教坏多少人

发表于 2020-05-03 分类于 商事

如果你也打算抢公章

不久前，当当网股东抢公章事件上了热搜，有些朋友受到启发，以为得公章者可得天下，萌生了通过抢公章搞政变夺权的想法。悟性很好，触类旁通。不过，有些事希望你能了解一下：

这抢夺印章可要分对象，您打算抢的是企业的章，那还好，至少不用坐牢，毕竟作为财物它不值什么钱；但您要是打国家机关印章的主意，可要三思而后行，这种升官发财的秘诀已经被写入现行《刑法》，不仅伪造不得，也抢不得。所以，下面我们要“抢”的章仅在企业印章的范畴内。

不坐牢不代表不能关

所谓“关几天”用行话说叫“限制人身自由的强制措施”，分刑事强制措施与行政强制措施，俗话“坐牢”指的就是因触犯刑律而被判处刑罚由监狱实施的那种，生活中很多“进去了”的情况可能只是行政拘留，比如强拿硬要公私财物的行为，会因为违反《治安管理处罚法》被处五日以上拘留甚至罚款，毕竟刻章也是有成本的。

静置的章能值几钱

印章的生命在于使用，因此社会生活中会有“萝卜章”的存在，只要能印出花儿来，就能达到为非作歹的目的了。如果一颗漂亮无瑕的公章只是静静地待在那儿，就被你抢走了，章作为章本身也是有尊严的，它在物权法的角度上，属于公司所有、具体由公司规定由何人占有的财物。当你违背公司意志占有公章，因此被脱离占有的保管人有权基于物权法上的占有制度，请求返还原物。当然这种“请求”一般要通过向法院提起诉讼来实现。

公章缘何炙手可热

公章是公司对外作意思表示的一种法定方式，另一种方式是法定代表人的签字。换言之，公章与法定代表人都可以对外代表公司意志，因此当与公司签署合约时，只要认准对方这公章，除非有强有力的相反证据排除公章的公示效力，否则即便用章人未获得公司授权，也可依法构成表见代理，发生的法律效果归属于公章指向的公司。这也是公章屡屡被抢的原因，抢章的人以为只要能够对外使用公章，就能据此实际控制公司。至于能否实现控制效果，一方面取决于交易相对方是否知情公章已经脱离公司控制的事实，另一方面取决于公司章程对公章保管人及其使用程序的具体规定。

公章被抢岂能善罢甘休

公章本身可能并不值钱，但那是公司对外签约的象征，不拿回来的后果不堪设想，它有可能成为不当持章者违法犯罪的工具，间接损害公司利益。

怎么拿回来？由谁来主张权利？因为公章被抢给公司造成实际损失，侵权人应当承担赔偿责任，这一种情形不同于要求侵权人返还印章的诉讼，是可以由股东代表受损公司提起诉讼的。但公司证照返还纠纷只能以公司为原告，以侵占人为被告诉至法院。鉴于代表公司主体意志的公章系本案讼争的标的，公司作为原告的意思表示机关即可以是法定代表人，如果恰巧法定代表人是被告，则需要公司股东会决议授权对外签字人。因此公章被抢的第一反应报警没错，报警的目的不仅仅希望民警来主持公道（公安只懂治安法），更多时候是为了留下公章脱离合法占有的证据，以备日后诉讼所需。

股东掌印越俎代庖

股东是公司的股东，是公司的所有者。但公司不是某一个股东所有的，是全体股东按份所有。公章是公司所有的财产，有时可能会被股东占有和使用，但那都基于环环相扣的授权。因此，当公司的公章被抢，不论抢章的是股东还是第三人，为公章做主的只能是公司，对公司享有不完整所有权的股东，不论是抢章还是公章被抢后的私力救济，都是越俎代庖的行为。除非该股东所持公司股份足以左右股东会决议。
